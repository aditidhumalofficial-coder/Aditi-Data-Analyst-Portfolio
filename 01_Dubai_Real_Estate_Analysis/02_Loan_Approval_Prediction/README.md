# Loan Approval Prediction Model

## 📌 Business Problem
Banks need to determine whether a loan application should be approved based on applicant financial details.

---

## 📊 Dataset Features
- Age
- Salary
- Loan Approval Status (Approved / Not Approved)

---

## 🤖 Model Used
Logistic Regression

---

## 🔍 Analysis Performed
- Feature selection
- Model training
- Coefficient interpretation
- Prediction on new data

---

## 📈 Business Impact
Helps financial institutions reduce risk by predicting loan approval probability.
