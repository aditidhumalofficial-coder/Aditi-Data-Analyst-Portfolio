# Dubai Real Estate Market Analysis

## 📌 Business Problem
Investors and buyers need insights into Dubai property prices to identify premium and high-growth areas.

---

## 📊 Dataset Features
- Area (sqft)
- Bedrooms
- Bathrooms
- Location
- Property Price

---

## 🔍 Analysis Performed
- Average price by location
- Price per square foot calculation
- Comparative location analysis
- Data visualization using bar charts

---

## 📈 Key Insights
- Palm Jumeirah has the highest property prices.
- JVC offers affordable housing options.
- Downtown Dubai and Marina show strong mid-high pricing trends.

---

## 🛠 Tools & Technologies
Python | Pandas | Matplotlib | Power BI

---

## 💡 Business Impact
This analysis helps investors identify profitable real estate opportunities in Dubai.
